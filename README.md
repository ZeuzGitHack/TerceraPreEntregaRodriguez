Web creada para compilar y compartir informacion sobre libros con el nombre "ProyectoBookMe".
La url de la app es: "AppBook"
Desde la barra de navegacion se puede acceder a la informacion de las tres clases de los modelos creados, a demas de al formulario de busqueda de libros
Primero sugiero navegar desde el boton "Inicio", y despues a los demas botones "Libros", "Autores", "Editoriales", en los que ya tienen algunos datos cargados.
Despues ingresar al boton "Buscar" con el formulario de busqueda, buscar cualquier titulo en la base de datos, o datos incorrectos
Para entrar a los formularios de las Clases y cargar datos a la base de datos, son con las siguientes url's:
'AppBook/formulario-libro/'
'AppBook/formulario-autor/'
'AppBook/formulario-editorial/'
